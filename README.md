# 📈 Sales Prediction System

A Machine Learning project that predicts product sales based on advertising spend across TV, Radio, and Newspaper channels.

---

## 🧠 Model Details

| Metric | Score |
|--------|-------|
| Algorithm | Linear Regression |
| R² Score | ~0.90 |
| RMSE | ~1.6 |

> *Run the script to see your exact scores*

---

## 📊 Dataset

Uses the classic `advertising.csv` dataset with features:
- **TV** — advertising spend on TV
- **Radio** — advertising spend on Radio
- **Newspaper** — advertising spend on Newspaper
- **Sales** — target variable (units sold)

---

## 🔁 Project Workflow

1. Load & explore dataset
2. Visualize feature vs sales relationships
3. Correlation heatmap analysis
4. Train/test split (80/20)
5. Train Linear Regression model
6. Evaluate with R² & RMSE
7. Visualize Actual vs Predicted Sales
